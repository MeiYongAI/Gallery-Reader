Gallery Readerv-

Modern multi-site reader (E-Hentai / ExHentai / nhentai / hitomi.la / Wnacg)

Core Features

  • Multi-site support
  • Real image loading progress
  • Smart image preloading
  • Reading progress memory

Supported Sites

  • E-Hentai / ExHentai
  • nhentai.net / nhentai.xxx
  • hitomi.la
  • Wnacg / 紳士漫畫

Install and Debug

  • Open chrome://extensions/
  • Enable Developer mode
  • Click "Load unpacked" and choose this project folder
  • After editing code, click the reload button on the extension card

Site Readers

hitomi.la and Wnacg now take over native reader entries and use native image lists plus gallery thumbnails.
